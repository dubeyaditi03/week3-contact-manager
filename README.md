# Contact Management System

## Project Description

This is a Contact Management System built using Python Functions and Dictionaries.

The project allows users to:

- Add contacts
- Search contacts
- View contacts
- Delete contacts
- Save contacts in JSON format

---

## Features

✔ Add New Contact

✔ View All Contacts

✔ Search Contact

✔ Delete Contact

✔ Save Contacts

✔ Load Contacts

✔ Dictionary Based Storage
